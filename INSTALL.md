# Hermes Agent Installation

![install](screenshots/02_hermes_docs.png)

## Linux/Mac/WSL
```bash
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
```

Verify:
```bash
hermes --version
hermes doctor
```

## Agent Desktop
1. Download from: https://github.com/NousResearch/hermes-agent/releases
2. Install & Launch
3. Connect to local instance or messaging gateway
