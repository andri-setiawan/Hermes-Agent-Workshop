# NotebookLM Automation

![notebooklm](screenshots/09_notebooklm.png)

## Setup
```bash
pip install notebooklm-py
```

## Usage
Collect PDFs and manage them in NotebookLM:

```python
# notebooklm_manager.py
from notebooklm import NotebookLM

client = NotebookLM()
# Upload PDFs from local corpus
# Create / update notebook with grounded context
```
